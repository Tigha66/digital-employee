## Day 1-2: Connection Phase

### Step 1: Email Setup
```
1. Client shares Gmail access (or gives app password)
2. Set up email filters in Gmail
3. Create labels: "Urgent", "Leads", "Follow-up"
4. Test: Send test email to see filters work
```

### Step 2: Calendar Setup
```
1. Share calendar with AI email address
2. Set availability hours
3. Create booking link (or use client's)
4. Set buffer time between meetings
```

### Step 3: AI Configuration
```
1. Input client's business info:
   - Business name
   - Services offered
   - Common questions & answers
   - Pricing (if applicable)
2. Set tone of voice (formal/casual)
3. Set working hours
```

---

## Day 3-4: Configuration Phase

### Step 4: Response Templates

Create these 5 templates:

#### 1. Initial Lead Response
```
Hi [Name],

Thanks for reaching out! I'd be happy to help with [service].

To get you the best [quote/solution], I need:
1. [Question 1]
2. [Question 2]
3. [Question 3]

I've scheduled a call with [Client] for [time]. Does that work?

Best,
[Business Name]
```

#### 2. Scheduling Confirmation
```
Hi [Name],

Your appointment is confirmed for [Date] at [Time].

[Client] will [action].

We'll send a reminder before the meeting.

See you soon!
```

#### 3. Follow-up Template
```
Hi [Name],

Just following up on our conversation about [topic].

[Client] mentioned you're interested in [details].

Would you like to [next action]?

Best,
[Business Name]
```

#### 4. No Response Follow-up
```
Hi [Name],

Just checking if you got my last message about [topic].

Let me know if you have questions!

Best,
[Business Name]
```

#### 5. Closing/Thank You
```
Hi [Name],

Thank you for [action]!

[Next steps / What happens next]

Looking forward to [outcome]!

Best,
[Business Name]
```

### Step 5: CRM Setup (If applicable)
```
1. Connect to client's CRM
2. Set up lead capture form
3. Create auto-tasks for follow-ups
4. Configure pipeline stages
```
